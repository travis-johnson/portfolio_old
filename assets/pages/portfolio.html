Homeshare is a home organization app with gamification features. It features a point system for completing chores and buying items from a household shopping lists as well as using an adaptive payments API from Paypal for bill/rent payment between housemates.

Created with: HTML5, SASS, Susy(framework), jQuery, AngularJS and Ruby on Rails.

BroToGo is a day planning chat-bot app for anyone looking to get out of the house and have fun. The user is able to search for local businesses or restaurants, weather, local breweries as well as nearby concerts. These features are connected to the Foursquare API, Dark Sky Weather API, BreweryDB API, and Jambase API. There are also a few built in replies to allow the chat-bot to become truly interactive. The app is responsive and displays in both mobile and desktop views.

Created with: HTML5, SASS, Susy(framework) and jQuery
